### This is a simple app for learning language

## Setup

* `git clone github.com/OlegVladimirovichOleynik/Flashcards_part2.git`
* `bundle install`
* `cp config/database.ci.yml config/database.yml`
* `rake db:setup`
* `rake db:seed`

## Tech
* Ruby on Rails 5
* Ruby 2.4
* Ansible for provisioning
